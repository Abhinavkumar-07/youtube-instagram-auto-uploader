# Contributing

Contributions, issues, and feature requests are welcome.

## Found a bug or have an idea?

Open an [issue](../../issues) — include what you expected vs what happened,
and any relevant error output (redact your own API keys/tokens first).

## Want to submit a change?

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-idea`
3. Make your changes
4. Test locally (see README setup) before opening a PR
5. Open a pull request describing what changed and why

## Ideas for contributions

- Support for additional platforms (TikTok, Facebook, LinkedIn)
- A dry-run mode that previews generated metadata without uploading
- Support for other free LLM providers beyond Groq
- Better error handling / retry logic for flaky network conditions
- Tests

No contribution is too small — typo fixes and doc improvements are just as
welcome as code.
